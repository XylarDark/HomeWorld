# Vision Board

This directory holds **vision, MVP, and world design** for HomeWorld, plus **prompt files** for generating a vision board (concept art, mood boards, AI image prompts). Content is organized into subdirectories below.

---

## Layout

| Subdirectory | Contents |
|--------------|----------|
| **[Core/](Core/)** | Central vision and scope: **VISION.md** (theme, campaign, moral system, day/night), **PROTOTYPE_SCOPE.md**, **STACK_PLAN.md**. |
| **[Prompts/](Prompts/)** | One file per theme for vision-board prompts: Aesthetics, Character_Creation, Gameplay_Mechanics, Scope_And_Deliverables, World_Building, World_Lore, World_Progression. Use as sources for concept art or AI prompts. |
| **[MVP/](MVP/)** | MVP and delivery: phased approach, tutorial plan, vertical slice checklists, Steam checklist, next-30-day window, last-session audit, MVP gap/scope docs. |
| **[Planetoid/](Planetoid/)** | Planetoid and world design: PLANETOID_DESIGN, PLANETOID_BIOMES, PLANETOID_PRIDE_MVP, PLANETOID_TO_REALITY_AND_WORLD_BUILDER, WORLD_BUILDER_AND_MODIFIERS. |
| **[Character/](Character/)** | Character creation and UI: **CHARACTER_GENERATION_AND_CUSTOMIZATION.md** (main menu, character screen, first-launch flow). |

---

## Entry points

- **Theme and scope:** [Core/VISION.md](Core/VISION.md), [Core/STACK_PLAN.md](Core/STACK_PLAN.md)
- **Vision-board prompts:** [Prompts/](Prompts/) (Aesthetics, Gameplay Mechanics, World Lore, etc.)
